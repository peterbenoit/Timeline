  3. Publication

    2018

    Release of updated Technical Standards Report on uniform criteria for system integration, entitled Definitions for Enterprise System Architecture . These definitions were developed collaboratively across departments and approved by the technical oversight committee.

  4. Research

    2016

    In collaboration with industry partners, the team publishes a report entitled Framework for Next-Generation Systems . It provides uniform criteria for implementation. These standards were developed with input from multiple stakeholders and represent industry consensus.

  5. Technology

    2014

    Initial system launch. The platform is built to allow organizations to collect, process, and share data through standardized APIs and data formats.
